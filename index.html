SlideCops with CNES

ISAE-SUPAERO, Toulouse

Development of an application using satellite data to estimate the damages of a landslide
The purpose of this project is to give decisionnal informations to the rescue team, in order to optimize the process
In order to do this, we needed to simulate the flow, and to do so, we've found the opensource software "r.avaflow".
This software must take in input :

- A DEM of the area (using Light Detection and Ranging - LiDAR data)
- The landslide source (OSM Satellite data using SAR and stéréo)
- The landslide volume (the amount of material that has been displaced)
- The friction parameters (which depend on the type of material and the slope)

And with all of that, we can simulate the flow and get the resulting flow path and impact area. Then, with an infrastructure detection system (using 2 possibles sources, LIDAR or OSM), we can identify which infrastructures are at risk and provide this information to the rescue team.
Also, we can identify which road can be used to evacuate the area.

We had the chance to present our solution at CNES, the French space agency, to have a feedback from them, which was very positive and encouraging for the future of this project. However, we are still in the early stages of development, we need to determine the marketizing of our solution, but we are confident that with the help of CNES and other partners, we can make this project a success and help save lives in the future.

Septembre 2025 - Now

Reforestation Tutored Project AIRBUS

ISAE SUPAERO, Toulouse

    The goal of this project is to develop a method to monitor the reforestation process using satellite data and key indicator, we started from the VERRA VM0027 methodo which quantifies the reduction in carbon dioxide (CO2) emissions due to decreased oxidation of soil organic material that occurs as a result of project activities. BUT the reduction in carbon dioxide isn't enought to determine a good monitoring, so we need to assure a good evolution of :

    - Ecosystem structure
    - Ecosystem functions
    - Captured Carbon

    Thanks to key indicators such as FSCI, LFCI for the ecosystem structure, and the GPP/NPP, EBI for the ecosystem functions.

    Also, we need to get the risk assessment following some key indicators that we are currently searching for. In addition, it's important to keep in mind the differents leakage risks, such as the market shifting leakage, the activity shifting leakage for now.

    This project is still in development, we are currently working on the most relevants indicators to monitor the reforestation process in the best way possible.
